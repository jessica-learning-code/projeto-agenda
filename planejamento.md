# Projeto-Agenda

#Objetivo

O objetivo desse projeto é criar uma simples agenda para facilitar o manuseio das atividades cotidianas.

#Funcionalidades

Suas funcionalidades são escrever suas tarefas diárias, agendar eventos futuros, calendário simples do mês mostrando feriados.

#Tempo de execução 

Tempo de execução em 3 dias.
